# Zendr

Browser-based peer-to-peer file sharing powered by WebRTC.

Self-hostable, privacy-focused, and requiring nothing more than a modern web browser.

> 🚧 **Early development**
>
> Zendr is currently under active development. The architecture, protocol, APIs, and roadmap are subject to change before the first stable release.

## Goals

- 🌐 Browser-based
- 🔒 Direct peer-to-peer file transfers
- 🏠 Self-hostable
- ⚡ Simple and fast
- 📂 Multi-file transfers

## Roadmap

- [x] Device discovery
- [ ] Transfer requests
- [ ] WebRTC peer connections
- [ ] LAN peer-to-peer file transfers
- [ ] Multi-file transfers
- [ ] Live transfer progress
- [ ] QR pairing
- [ ] Rooms
- [ ] Internet peer-to-peer file transfers
